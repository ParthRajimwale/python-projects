#Python Calculator 

#About
This is a simple calculator built using Python. It performs basic mathematical operations like addition,subtraction,multiplication,division and powers.

#Features
-Takes 2 numbers and operation to be performed as input from the users
-Supports Aaddition,Subtraction,Multiplication,Division and Powers
-Handles division by 0
-Runs until the user chooses to exit

#What I learned
-The use of conditional statements(if-elif-else)
-Use of loops(while loop)
-Preventing errors such as division by zero
-Using the lower() function
